#+TITLE: Canti Post Mitering Fixture

[[./canti-post-mitering-fixture.png]]

Hold canti posts for mitering. Allow the correct miter angle to be set relative to its spring holes.

One hole is for standard canti posts. The other is for Mafac/Rene Herse centerpull posts.
